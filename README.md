# Vim-Setting

## OS, CPU & GPU configuration
### Ubuntu version
first check your Ubuntu version, choose one of them
```bash
cat /etc/issue
lsb_release -a
cat /etc/os-release
```

### CPU configuration
```bash
# version
lscpu
```

## How to set vim
### If you use, 22.04 Ubuntu or upper version,
```bash
bash install.sh
```

### If you use, 20.04 Ubuntu or under version,
```bash
bash install-o.sh
```
and next
```bash
bash setting.sh
```
